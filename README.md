# SUMO_MINI_MICRO
Repositorio para proyectos de Robots Categoría _SUMO, Todos los archivos tienen derechos reservados y no pueden distribuirse a terceros, solo para uso personal.

En el repositorio encontrarás 2 modelos de minisumo profesional y 1 modelo de minisumo amateur para poder usarlo en cualquier competencia ya que cumple con los estandares internacionales.

El modelo de placa gratutia que podes usar es la orion 3.0 una placa pensada para minisumo con motores N20 super factible el uso en esos modelos, para motores más exigentes se recomiendan usar otros modelos con MOSFETS DRIVERS que soportan más amperaje y tensión.
